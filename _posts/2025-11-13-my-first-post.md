### Getting Started

To create a new blog post, simply:

1. Create a new Markdown file in the `_posts/` directory
2. Follow the naming convention: `YYYY-MM-DD-title.md`
3. Add the front matter (YAML) at the top
4. Write your content in Markdown

### Why Jekyll?

Jekyll is perfect for:
- **Bloggers**: Focus on writing, not maintenance
- **Developers**: Version control your content with Git
- **Performance**: No database needed, just static HTML
- **Security**: No server-side code to exploit
- **Free Hosting**: Deploy to GitHub Pages at no cost
