# SEO Checklist for Your Jekyll Blog

Complete checklist to ensure your blog is fully optimized for search engines.

## Pre-Launch Checklist

### Site Configuration
- [ ] Update site title in `_config.yml`
- [ ] Update site description in `_config.yml`
- [ ] Set correct site URL in `_config.yml`
- [ ] Add author name and email
- [ ] Add social media usernames
- [ ] Set up Google Analytics ID
- [ ] Create favicon and add to `assets/images/`
- [ ] Create apple-touch-icon and add to `assets/images/`

### Technical SEO
- [ ] Verify site is mobile-responsive
- [ ] Test on multiple devices
- [ ] Check page load speed with PageSpeed Insights
- [ ] Ensure HTTPS is enabled (GitHub Pages default)
- [ ] Create XML sitemap (auto-generated)
- [ ] Create robots.txt (included)
- [ ] Set up 404 page (optional)
- [ ] Enable gzip compression

### Content Structure
- [ ] Create homepage with compelling content
- [ ] Create About page with author bio
- [ ] Create Contact page with contact form
- [ ] Create Blog listing page
- [ ] Create Tags page
- [ ] Create Categories page
- [ ] Add internal navigation links
- [ ] Ensure proper heading hierarchy

### Social Media
- [ ] Add Open Graph meta tags (included)
- [ ] Add Twitter Card tags (included)
- [ ] Test social sharing with [Social Share Preview](https://www.socialsharepreview.com/)
- [ ] Add social media links to footer
- [ ] Create social media accounts

## Per-Post Checklist

### Before Publishing

#### Title & Description
- [ ] Title is 50-60 characters
- [ ] Title includes primary keyword
- [ ] Title is compelling and clickable
- [ ] Meta description is 150-160 characters
- [ ] Description includes primary keyword
- [ ] Description is action-oriented

#### Content Quality
- [ ] Content is original and valuable
- [ ] Content is at least 1000+ words (recommended)
- [ ] Content is well-structured with headings
- [ ] One H1 per page (post title)
- [ ] Use H2 and H3 for subheadings
- [ ] Content is easy to read (short paragraphs)
- [ ] Include bullet points and lists
- [ ] Use bold and italic for emphasis

#### Keywords & SEO
- [ ] Primary keyword appears in title
- [ ] Primary keyword appears in first 100 words
- [ ] Primary keyword appears in meta description
- [ ] Primary keyword appears in H1 and H2 tags
- [ ] Use related keywords naturally
- [ ] Keyword density is 1-2% (not over-optimized)
- [ ] No keyword stuffing

#### Links
- [ ] Include 3-5 internal links to related posts
- [ ] Use descriptive anchor text
- [ ] Link to authoritative external sources
- [ ] Use `rel="noopener noreferrer"` for external links
- [ ] Check all links are working

#### Images
- [ ] Include at least one featured image
- [ ] Image is optimized (under 200KB)
- [ ] Image has descriptive filename
- [ ] Image has alt text (descriptive)
- [ ] Image is relevant to content
- [ ] Use high-quality images
- [ ] Consider using WebP format

#### Front Matter
- [ ] Layout is set to `post`
- [ ] Title is filled in
- [ ] Date is correct
- [ ] Author is specified
- [ ] Categories are relevant (1-3)
- [ ] Tags are relevant (3-5)
- [ ] Featured image path is correct
- [ ] Image alt text is descriptive
- [ ] Excerpt is compelling (150-160 chars)
- [ ] Reading time is estimated

#### Metadata
- [ ] Post has excerpt for search results
- [ ] Post has reading time estimate
- [ ] Post has author information
- [ ] Post has publication date
- [ ] Post has last modified date (if updated)

## Post-Launch Checklist

### Search Engine Submission
- [ ] Submit sitemap to Google Search Console
- [ ] Submit sitemap to Bing Webmaster Tools
- [ ] Verify site ownership in Google Search Console
- [ ] Verify site ownership in Bing Webmaster Tools
- [ ] Request indexing of homepage
- [ ] Request indexing of first post

### Monitoring & Analytics
- [ ] Set up Google Analytics
- [ ] Set up Google Search Console
- [ ] Monitor search performance
- [ ] Monitor traffic sources
- [ ] Monitor user behavior
- [ ] Check for crawl errors
- [ ] Monitor keyword rankings

### Content Promotion
- [ ] Share on social media
- [ ] Share in relevant communities
- [ ] Add to RSS feed
- [ ] Email to subscribers
- [ ] Add to content calendar
- [ ] Plan follow-up posts

## Ongoing Maintenance

### Monthly Tasks
- [ ] Review Google Search Console data
- [ ] Check for broken links
- [ ] Update old posts with new information
- [ ] Monitor keyword rankings
- [ ] Analyze top-performing content
- [ ] Plan new content

### Quarterly Tasks
- [ ] Audit site structure
- [ ] Review and update meta descriptions
- [ ] Check page load speed
- [ ] Update internal links
- [ ] Review and improve images
- [ ] Check for duplicate content

### Annually Tasks
- [ ] Full site audit
- [ ] Update outdated content
- [ ] Review and refresh old posts
- [ ] Analyze competitor strategies
- [ ] Plan content strategy for next year
- [ ] Update site design if needed

## SEO Best Practices Summary

### Content
✅ Write for humans first, search engines second
✅ Create original, valuable content
✅ Use natural language and keywords
✅ Keep content fresh and updated
✅ Use proper heading hierarchy

### Technical
✅ Ensure fast page load speed
✅ Mobile-responsive design
✅ Use HTTPS
✅ Create XML sitemap
✅ Use structured data (JSON-LD)

### Links
✅ Build quality backlinks
✅ Use internal linking strategy
✅ Use descriptive anchor text
✅ Avoid broken links
✅ Link to authoritative sources

### User Experience
✅ Easy navigation
✅ Clear call-to-actions
✅ Readable content
✅ Accessible design
✅ Fast performance
